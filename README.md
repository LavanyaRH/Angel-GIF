# Simple Image-to-GIF Converter

This is my first Python project! I built this to learn how to handle external libraries and automate a creative task.

## Features
* **Format Support:** Works with both `.png` and `.jpeg` files.
* **Auto-Resizing:** Automatically resizes images to ensure the GIF doesn't crash due to dimension mismatches.
* **Custom Speed:** Easily adjustable frame duration.

## What I Learned
* How to manage Python environments and install packages via `pip`.
* Handling `ValueErrors` related to image "shapes" (dimensions).
* Basic file path management in Windows.

## Libraries Used
* `imageio`
* `Pillow` (PIL)
* `numpy`
